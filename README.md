# 📝 Todo Backend API

This repository contains the backend for a **User-Specific Todo Application** built using the **MERN stack**.
The API allows users to securely create, read, update, and delete their personal todos with **JWT-based authentication**.

---

# 🚀 Tech Stack

* **Node.js**
* **Express.js**
* **MongoDB**
* **Mongoose**
* **JWT Authentication**
* **bcrypt (Password Hashing)**
* **Zod (Input Validation)**
* **dotenv (Environment Variables)**

---

# 📌 Features

* User Signup and Login
* Secure Password Hashing with **bcrypt**
* **JWT-based Authentication**
* Create Todo
* Update Todo
* Delete Todo
* Fetch user-specific todos
* Input validation using **Zod**
* Protected routes using middleware

---

# 📦 Installation

Clone the repository

```
git clone https://github.com/yourusername/todo-backend.git
```

Move into the project directory

```
cd todo-backend
```

Install dependencies

```
npm install
```

---

# ▶️ Running the Server

Start the server

```
node index.js
```

or

```
npm start
```

The server will run at:

```
http://localhost:3000
```
